Воркер.Директор - как видно из названия, это рограмма, которая устанавливается на компьютер директора и является сердцем системы Воркер. С помощью программы Воркер.Директор осуществляеется:

  • настройка системы учета и контроля времени
  • создание и управление Агентами - программами, которые собирают статистику по сотрудникам
  • просмотр собранной статистики по сотрудникам

Схема работы системы

Скачиваем с сайта программу Воркер.Директор
Программа Воркер.Директор устанавливается на компьютер директора.
При первом запуске требутся получить пин-код активации. Для этого нужно ввести омер телефона. В ближайшее время придет СМС, введите код в соответсвущее поле
Придумайте пароль. Этот пароль необходим для входа в программу Воркер.Директор
После запуска, программа предложит создать Воркер.Агента. Созданного агента нужно записать на Flash-накопитель
Программа Воркер.Агент устанавливается на компьютеры сотрудников. После установки программа начинает собирать данные о действиях производимых на компьютере.
Данные шифруются индивидуальным ключом и отправляются в облачное хранилище.
В облаке данные хранятся в зашифрованном виде.
По заросу из программы Воркер.Директор, данные из облака в зашифрованном виде передаются на компьютер директора.
На компьютере директора данные, поступающие из облака, расшифровываются перед выводом на экран.
